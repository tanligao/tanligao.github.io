在需要把自己写的 python 脚本供其他同事使用时，需要将脚本打包成 exe，就能够在其他没有安装 python 环境的电脑上运行，打包 python 的工具有：py2exe、cx_Freeze、py2app、pyinstaller 等。

我用的是相对来说比较方便处理的 pyinstaller 工具，记录下打包过程中遇到的包引用错误。

pyinstaller 简介

pyinstaller 官网操作文档，pyinstaller 工具能够在 Windows、Mac OS X 和 Linux 上将 python 脚本或者 PyQt 打包成可执行文件，可执行文件可以在其他没有 python 环境的电脑上运行，不过有时会出现莫名的问题。

pyinstaller 打包的基本原理：

  • 首先分析出 python 代码使用 import 导入依赖的包，注意包引入的递归关系，如果有使用第三方包则需要添加到 script.spec 中的 sys.path 变量中，script.spec 为打包命令生成可修改的配置文件
  • 把 *.py 文件编译成 *.pyc 文件
